# React 19의 혁신적인 새 기능들

React 19가 드디어 출시되었습니다! 이번 업데이트는 개발자 경험을 크게 향상시키는 여러 기능들을 포함하고 있습니다. 주요 변경사항을 하나씩 살펴보겠습니다.

## 1. React Compiler

React 19의 가장 큰 변화는 새로운 컴파일러입니다. 이제 `useMemo`, `useCallback`, `React.memo`를 수동으로 사용할 필요가 없습니다.

### Before (React 18)

```javascript
function TodoList({ todos, filter }) {
  const filteredTodos = useMemo(
    () => todos.filter(todo => todo.status === filter),
    [todos, filter]
  );

  const handleClick = useCallback((id) => {
    // handle click
  }, []);

  return (
    // JSX
  );
}
```

### After (React 19)

```javascript
function TodoList({ todos, filter }) {
  // 컴파일러가 자동으로 최적화!
  const filteredTodos = todos.filter(todo => todo.status === filter);

  const handleClick = (id) => {
    // handle click
  };

  return (
    // JSX
  );
}
```

## 2. Server Components 개선

서버 컴포넌트가 더욱 강력해졌습니다:

```javascript
// server-component.js
async function BlogPost({ id }) {
  // 서버에서 직접 데이터베이스 쿼리
  const post = await db.posts.findById(id);

  return (
    <article>
      <h1>{post.title}</h1>
      <Content markdown={post.content} />
    </article>
  );
}
```

## 3. Actions

폼 처리가 획기적으로 간단해졌습니다:

```javascript
function ContactForm() {
  async function submitForm(formData) {
    "use server";

    const email = formData.get("email");
    const message = formData.get("message");

    await sendEmail(email, message);
  }

  return (
    <form action={submitForm}>
      <input name="email" type="email" required />
      <textarea name="message" required />
      <button type="submit">전송</button>
    </form>
  );
}
```

## 4. use() Hook

Promise와 Context를 더 유연하게 사용할 수 있습니다:

```javascript
import { use } from "react";

function Comments({ commentsPromise }) {
  // Promise를 직접 사용
  const comments = use(commentsPromise);

  return (
    <ul>
      {comments.map((comment) => (
        <li key={comment.id}>{comment.text}</li>
      ))}
    </ul>
  );
}
```

## 5. Document Metadata

메타데이터 관리가 컴포넌트 내에서 가능해졌습니다:

```javascript
function BlogPost({ post }) {
  return (
    <>
      <title>{post.title}</title>
      <meta name="description" content={post.excerpt} />
      <meta property="og:title" content={post.title} />

      <article>
        <h1>{post.title}</h1>
        <div>{post.content}</div>
      </article>
    </>
  );
}
```

## 6. 향상된 Error Boundary

에러 처리가 더욱 세밀해졌습니다:

```javascript
function ErrorBoundary({ children, fallback }) {
  return (
    <ReactErrorBoundary
      fallback={fallback}
      onError={(error, errorInfo) => {
        // 에러 로깅
        console.error("Error caught:", error);
      }}
      onReset={() => {
        // 리셋 로직
      }}
    >
      {children}
    </ReactErrorBoundary>
  );
}
```

## 7. Suspense 개선사항

Suspense가 더 많은 시나리오를 지원합니다:

```javascript
function App() {
  return (
    <Suspense fallback={<Loading />}>
      <AsyncComponent />
      <Suspense fallback={<CommentsLoading />}>
        <Comments />
      </Suspense>
    </Suspense>
  );
}
```

## 8. 성능 개선

React 19는 내부적으로 많은 최적화가 이루어졌습니다:

- **번들 크기 감소**: 약 10% 감소
- **초기 로딩 속도**: 20% 향상
- **메모리 사용량**: 15% 감소
- **리렌더링 최적화**: 자동 배칭 개선

## 마이그레이션 가이드

### 1단계: 의존성 업데이트

```bash
npm install react@19 react-dom@19
```

### 2단계: 컴파일러 설정

```javascript
// vite.config.js
export default {
  plugins: [
    react({
      compiler: true, // React Compiler 활성화
    }),
  ],
};
```

### 3단계: 불필요한 최적화 제거

- `useMemo`, `useCallback` 제거
- `React.memo` 제거 (필요한 경우만 유지)

## 실전 예제: Todo 앱

React 19의 기능을 활용한 Todo 앱:

```javascript
"use client";

function TodoApp() {
  async function addTodo(formData) {
    "use server";

    const todo = formData.get("todo");
    await db.todos.create({ text: todo });
  }

  return (
    <>
      <title>My Todo App</title>

      <form action={addTodo}>
        <input name="todo" required />
        <button>추가</button>
      </form>

      <Suspense fallback={<div>Loading...</div>}>
        <TodoList />
      </Suspense>
    </>
  );
}
```
